# 引用方法

---

CSS 文件直接在<head>节点引入

```
<head>
    <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width; initial-scale=0.2" />

    <title>海益投资交易管理系统</title>
    <link rel="shortcut icon" href="/WebSource/Common/imgs/logo.png" type="image/x-icon" />

    <link href="/WebSource/Common/css/bootstrap.css" rel="stylesheet">  
    <link href="/WebSource/Common/css/all.css" rel="stylesheet">
    <link href="/WebSource/Common/css/font-awesome.min.css" rel="stylesheet">
    <link href="/WebSource/Common/css/Site.css" rel="stylesheet">
    <link href="/WebSource/Common/css/codemirror.css" rel="stylesheet">
    <link href="/WebSource/Common/css/show-hint.css" rel="stylesheet">
    <link href="/WebSource/Common/css/3024-night.css" rel="stylesheet">
    <link href="/WebSource/Common/css/bootstrap-multiselect.css" rel="stylesheet">
    <link href="/WebSource/Common/css/jquery.gridster.min.css" rel="stylesheet">


    <link href="/WebSource/Common/css/joint/joint.css" rel="stylesheet">
    <link href="/WebSource/Common/css/joint/rappid.min.css" rel="stylesheet">

</head>
```

```
    <script src="/WebSource/Common/js/jquery.js"></script>
    <script src="/WebSource/Common/js/bootstrap.js"></script>
    <script src="/WebSource/Common/js/box.debug.js"></script>
    <script src="/WebSource/Common/js/boxExtensions.debug.js"></script>
    <script src="/WebSource/Common/js/kendoui/kendo.all.js"></script>
    <script src="/WebSource/Common/js/kendoui/cultures/kendo.messages.zh-CN.min.js"></script>
    <script src="/WebSource/Common/js/kendoui/cultures/kendo.culture.zh-CHS.min.js"></script>
```
